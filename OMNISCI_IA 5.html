📖 Tutorial completo — Netlify (link gratuito e enviável)

1
Acesse github.com e clique em Sign up. Coloque seu e-mail, crie uma senha e confirme pelo e-mail.
2
Depois de entrar no GitHub, clique no + no topo → New repository. Em "Repository name" escreva omnisci. Clique em Create repository.
3
Na página do repositório, clique em "uploading an existing file". Arraste o arquivo OMNISCI_IA.html para lá. Antes de salvar, renomeie para index.html clicando no nome do arquivo. Depois clique em Commit changes.
4
Acesse netlify.com → clique em Sign up → escolha Continue with GitHub → autorize. Isso conecta os dois.
5
No painel do Netlify, clique em Add new site → Import an existing project → GitHub → selecione o repositório omnisci → clique em Deploy site.
6
Em 1 minuto aparece seu link: omnisci-xxxxx.netlify.app
Para personalizar o nome: vá em Site settings → Change site name → escreva omnisci-seu-nome.
Para atualizar depois: só trocar o arquivo no GitHub — o Netlify atualiza automaticamente!
⚠️ Seus códigos secretos já estão dentro do arquivo. Quem acessar o link usa como usuário normal. Só você tem os códigos.
